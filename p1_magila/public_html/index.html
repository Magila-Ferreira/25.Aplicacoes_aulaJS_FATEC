QUESTÃO 6

Explique o motivo para posicionar os códigos escritos em Javascript em um arquivo separado do código HTML.

Separar os tipos de código HTML e JS facilita:
1. Leitura do código (código limpo);
2. Manutenção do código;
3. Organização;
4. Separação das funcionalidades de cada parte do Script;
5. Carregar o JS após o HTML, para não haver erro de execução (defer);
